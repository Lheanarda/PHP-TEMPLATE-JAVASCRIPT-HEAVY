USAGE GUIDE
1. Folder Asset untuk Library, gambar, video, dll.
2. Ubah config untuk database params sesusi koneksi
3. Semua logic database ada di database-helper.php . Contoh pembuatan query seperti pada sample
4. Simpan Reusable layouts di folder layouts
5. Untuk panggil layout/asset sudah ada di config
6. Pemanggilan query di view bisa dilihat di contoh view index.php
